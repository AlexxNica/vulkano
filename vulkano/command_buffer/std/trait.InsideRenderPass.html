Required Methods

fn current_subpass(&self) -> u32

Returns the number of the subpass we're in. The value is 0-indexed, so immediately after calling begin_render_pass the value will be 0.

The value should always be strictly inferior to the number of subpasses in the render pass.

fn secondary_subpass(&self) -> bool

If true, only secondary command buffers can be added inside the subpass. If false, only inline draw commands can be added.
